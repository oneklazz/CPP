## Результаты

|    N    | LineList (сек) | Array (сек) |
|---------|----------------|-------------|
| 1000    |    0.000057    |   0.000261  |
| 5000    |    0.000250    |   0.005916  |
| 10000   |    0.000360    |   0.019047  |
| 50000   |    0.000750    |   0.219087  |
| 100000  |    0.001438    |   0.826314  |
| 500000  |    0.008067    |   21.1067   |
| 1000000 |    0.017075    |   84.7442   |

## Вывод

* `LineList` работает быстро, время растет почти линейно
* `Array` сильно тормозит на больших N из-за сдвига элементов
* Для задачи Иосифа (частое удаление из середины) `LineList` подходит лучше
