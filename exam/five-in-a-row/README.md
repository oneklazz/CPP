## Установка

**Вариант 1:**

[Скачать папку five-in-a-row](https://download-directory.github.io/?url=https://github.com/oneklazz/CPP/tree/main/exam/five-in-a-row)

**Вариант 2:**
```bash
git clone --filter=blob:none --no-checkout https://github.com/oneklazz/CPP.git
cd CPP
git sparse-checkout set exam/five-in-a-row
git checkout
```

## Сборка и запуск

```bash
cd five-in-a-row
make
make run
```
